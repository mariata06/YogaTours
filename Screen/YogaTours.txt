C:\Users\mas\Desktop\PSD_шаблон\YogaTours> npm uninstall jquery --save, жмем enter

Получаем в файле package.json пустое поле после удалившегося Jquery.  "dependencies": {}
Пустое поле можем удалить и сохранить файл.

Создаем папку SASS внутри папки Yogatour : mkdir sass

находясь в папке SASS создаем файл main.scss: cd sass : type nul > main.scss

В файле package.json создаем script: 
"scripts": {
    "compile:sass": "node-sass sass/main.scss css/style.css"
  } . Сохраняем.

После этого в командной строке запускаем компилятор: npm run compile:sass
